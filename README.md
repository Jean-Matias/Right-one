# Right-1 Assembly LLC Website

Static multi-page lead-generation website for Right-1 Assembly LLC.

## Pages

- `index.html` - homepage
- `services.html` - residential and commercial services
- `gallery.html` - filterable real project gallery
- `about.html` - company story and process
- `contact.html` - call, text, email, and quote request form

## Run

Open `index.html` in a browser or host the folder on any static web host.

The site uses public Right-1 Assembly LLC project image URLs from the existing company gallery. If local image files are provided later, replace the remote `https://right1assemblyllc.com/wp-content/uploads/...` URLs with local asset paths.
